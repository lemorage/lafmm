# LAFMM

**Livermore's Anticipating Future Movements Map** вЂ” an interactive terminal tool that implements Jesse Livermore's six-column price recording system from *How to Trade in Stocks* (1940).

You maintain CSV files with daily prices. The tool renders Livermore's 18-column map вЂ” column placement, pivotal points, trend confirmations, trading signals, Key Price, and market trend вЂ” all computed from the original rules.

Both human-readable (interactive TUI) and agent-readable (structured markdown). Point any LLM agent at `~/.lafmm/` and it knows what to do.

## Quick Start

```bash
uv sync
uv run lafmm
```

First run creates `~/.lafmm/` with everything you need. Then:

```bash
cd ~/.lafmm
# let your favorite agent work here
claude          # or codex, or any LLM agent
```

The agent reads `AGENT.md`, navigates `cache/`, and analyzes your market state. You add price data, the agent does the rest.

## How It Works

### For Humans

```bash
uv run lafmm       # launches interactive TUI
```

Navigate with arrow keys, `Enter` to drill in, `Esc` to go back, `q` to quit. Three levels: market dashboard в†’ group (18-column map) в†’ individual stock.

### For Agents

```bash
cd ~/.lafmm
cat cache/market.md               # market overview
cat cache/semis/group.md          # group detail with 18-col map
cat cache/semis/NVDA.md           # individual stock sheet
```

Structured markdown, self-contained per file. The agent reads `AGENT.md` for identity and domain knowledge, `cache/` for state, `skills/` for workflows.

### Adding a sector

Create a folder in `data/` with a `group.toml` and ticker directories:

```
~/.lafmm/data/semis/
в”њв”Ђв”Ђ group.toml
в”њв”Ђв”Ђ NVDA/
в”‚   в””в”Ђв”Ђ 2026.csv
в”њв”Ђв”Ђ AVGO/
в”‚   в””в”Ђв”Ђ 2026.csv
в””в”Ђв”Ђ AMD/             # tracked (optional, auto-discovered)
    в””в”Ђв”Ђ 2026.csv
```

**group.toml:**

```toml
name = "Semiconductors"
leaders = ["NVDA", "AVGO"]
swing_pct = 8.0
confirm_pct = 4.0
```

**CSV files** вЂ” one row per trading day:

```csv
date,price
2025-01-02,130.00
2025-01-03,133.00
```

The filename is the ticker. If it matches a leader name, it's a leader. Any other CSV is auto-discovered as a tracked stock вЂ” gets its own engine but doesn't affect Key Price.

## The System

### The Six Columns

```
| SecRally | NatRally | UPTREND | DNTREND | NatReac | SecReac |
|  pencil  |  pencil  |  BLACK  |   RED   | pencil  | pencil  |
```

- **Upward Trend** (green) вЂ” confirmed uptrend
- **Downward Trend** (red) вЂ” confirmed downtrend
- **Natural Rally / Reaction** (gray) вЂ” significant counter-moves in either direction
- **Secondary Rally / Reaction** (gray) вЂ” indecisive moves that haven't reached prior levels

### The 18-Column Map

Livermore tracked **groups** вЂ” two leader stocks per industry:

```
| Leader A (6 cols) | Leader B (6 cols) | Key Price (6 cols) |
```

**Key Price** = A + B combined, with doubled thresholds (swing=12, confirm=6). Group trend comes from Key Price, not individual stocks.

> *"There is danger of being caught in a false movement by depending upon only one stock."*

### Signals

| Signal | Rule | Meaning |
|--------|------|---------|
| BUY (resumption) | 10(a) | Uptrend resumed through prior pivot |
| BUY (reversal) | 10(d) | Downtrend failed |
| SELL (resumption) | 10(c) | Downtrend resumed through prior pivot |
| SELL (reversal) | 10(b) | Uptrend failed |
| DANGER: Up Over | 10(e) | Rally peaked near pivot but failed, then retreated |
| DANGER: Dn Over | 10(f) | Reaction bottomed near pivot but held, then bounced |

### Market Trend

\>60% of groups bullish в†’ market bullish. >60% bearish в†’ market bearish. Otherwise neutral.
